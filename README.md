# ML_S7_Eye_Classifier
this is a web app that Classify eye piscture and tell you if it is normal or have a deasese and wich one.

Please, run the requirement.txt befor running the project.
To do so on the terminal, run the following command (be carefull with the path, "/path/to/" is not necessary if youre cd is on the project folder): 

pip install -r /path/to/requirements.txt
python /path/to/WebApp_eyeClassifier.py
